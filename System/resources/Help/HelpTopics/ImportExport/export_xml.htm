Export as XML

Below, in figure 1., is an example of the XML Export dialog window. (see the Overview, for help on how to access this option).

N.B. The exported nodes and links will be taken from the current view, so make sure the current view is the one you wish to export from.


Figure 1. The XML Export dialog


There are five export option you can set:


Option

Purpose
Export all nodes on current map f this option is selected, it will only export all the node in the current map.

Export selected nodes only

If this option is selected, it will only export the node which have been selected in the current map.

Export current map depth only

If this option is selected, the export will only take the nodes at the current level of the map.

Export current map depth to full

If this option is selected, it will export all maps with their contents nodes, and their contents node etc, to the full depth of the map tree
Include Meeting and Media Index Data Additional information for the Memetic, (http://www.memetic-vre.net), project Meeting recoding options
Export to Zip Archive with images and Referenced files If you choose this option any referenced local files will be included in the export (in a subfolder called 'references') and all the html and additional files and directories will be placed in a zip file.

All references and images included in the zip file are stored in the in the path structure 'Linked Files/<current database name/<filename>'. The paths in the XML are changed accordingly.

Include your Stencils and Link Groups in Zip Archive This option is only available if you export to zip file, and allows you to include the stencil and link group data with the export.

Once you have completed the export options, press the 'Export' button. A file browser dialog window will open, (see figure 2 below), and you will need to locate the directory, and enter the file name you wish to export to.


Figure 2. The Export File Browser dialog


Once you have entered a file name, either html, or zip (if exporting to a zip archive), press the 'Save' button to begin the export. You will see a progress bar showing you how the export is progressing.
