The Node Right-Click Menu

There are different right-click node menus for maps and lists.

The Map node right-click menu:



Menu option

Allows you to

For more information

Contents

Open the current node to view and edit the Label, Detail, and other information.

See Node Contents and Properties

Change Type To...

Change the type of the current node, or all selected nodes, to node type selected from menu.

See Node Types

Embed into New Map

Create a new map node linked to the current node. The new map is then opened and an embeded version of the current node is placed inside.

 
Create Internal Reference Node

This option creates a node that is an internal reference node pointing at this node in this view. You can then cut and paste the internal reference node in another view. When the internal reference node is double-clicked it will open this view and highlight this node.

An internal reference node can be thought of as a
'GO TO' node which acts as a pointer to another node in a specific view.

see also Internal Reference Nodes
Send Node to Inbox of

This is part of the group working funtionality. It opens a dialog listing all other users for this project. When you select one or more users it creates an internal reference node in those users' inbox pointing to this node in this view.

Note: You can also add internal reference nodes to your own inbox using this option. So in a single user system your inbox could be used as another bookmarks list.

see Group Working
Apply Format to All Intances This option applies the formatting of this node, to all its embdeded node presences in other views.  
Apply Format to Full Depth This option only appears if the node is a view. This option applies the formatting of this node to all child nodes in this map, and may view inside those etc.. all the way down the view tree.
This is very powerful so be careful when using.		  
References... This option only appears if the node is a view and the view contains any reference nodes. It will lead to a submenu listing those references in order of reference type. When you select on it will open the reference with the appropriate external application.  
Copy

Copy a node, selection of nodes, link, selection of links, or text from a node Label or Detail.

When you Copy something, it is placed on your computer’s clipboard, available for subsequent actions. It is not removed from its current location.

Note: Subsequent Cut or Copy commands will replace whatever is on the clipboard.

 

Cut

Cut (remove) a node, selection of nodes, link, selection of links, or text from a node Label or Detail.

When you Cut something, it is removed from the current view or node and placed on your computer’s clipboard, available for subsequent actions. Cut nodes are placed in the Trash Bin, if they are not present in any other view.

Note: Subsequent Cut or Copy commands will replace whatever is on the clipboard.

 

Delete

Remove the current node from the current view. It is not placed on your computer’s clipboard. Deleted nodes are placed in the Trash Bin, if they are not present in any other view.

To retrieve items deleted in error, choose “Undo” from the Edit menu, or the main toolbar

 

Import

This option gives you import choices.

See Import / Export Overview

Export

This option gives you export choices.

See Import / Export Overview
Search Google Opens a browser window and runs a Google search on the label of the current node.  

Shortcut

Creates Shortcut nodes for all currently selected nodes.

See Creating Shortcuts

Clone

Creates Clone nodes for all currently selected nodes.

See Creating Clones

Delink

Remove all links pointing to or from the current node. A handy way to select and remove all links to a particular node.

See Removing Links

Bookmark Node

at this node to your bookmarks list.

See The Bookmarks Menu

Move Detail into Label

Move the detail text for all selected nodes, (or this node, if none selected), into their label fields

 

Move Label into Detail

Move the label text for all selected nodes, (or this node, if none selected), into the end of the first detail window.

 
Readers This is part of the group working functionality. This opens list dialog showing all the users who have viewed this node. see Group Working
Mark Seen

Mark the selected node as 'seen'. This is part of the group working functionality.

When other users add or edit nodes, you see them displayed with a red border to indicate that you should look at them as they have changed / are new. This option allows you to mark the selected node as seen. This will remove the red border from the node.

see Group Working
Mark Unseen

Mark the selected node as 'unseen'. This is part of the group working functionality.

see Group Working

Views

See which views, (including the current view), contain the current node.

See Views

Properties

Inspect the properties of the current node.

See Node Contents and Properties
