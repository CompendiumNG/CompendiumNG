Mark Seen

Mark the selected node as 'seen'. This is part of the group working functionality.

When other users add or edit nodes, you see them displayed with a red border to indicate that you should look at them as they have changed / are new. This option allows you to mark the selected node as seen. This will remove the red border from the node.

see Group Working
Mark Unseen

Mark the selected node as 'unseen'. This is part of the group working functionality.

see Group Working
Mark Seen All

This option only appears if you right-click on a view in the list. Mark the selected view node and all its contents nodes as 'seen'. This is part of the group working functionality.
