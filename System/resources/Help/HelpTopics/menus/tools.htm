The Tools Menu

(keyboard shortcut: Alt-T)



Menu option

Allows you to:

For more information:

User Manager
(keyboard shortcut:
Alt-T-U)

Administer users for the current database.

See User Administration.

Linked Files Browser
(keyboard shortcut:
Alt-T-L)

This opens the File Browser window which lets you manage your linked files.

See Linked Files Browser.
Templates (keyboard shortcut:
Alt-T-T)		 The First menu item friom this menu will me a link to the Template Help page. This remainder of this menu is dynamically created from the contents of your Compendium Templates folder. For each subfolder a submenu is created and any files are listed as menu items. You can have folders within folders. If you select a template file menu item, that template will be imported into the current open view.

See Templates.
memetic (keyboard shortcut:Alt-T-M) This opens the memetic submenu holding two options:

'Manual Stop', to stop a memetic meeting recording/replay if the Meeting Manager fails to do so.

'Manual Upload', this opens the Meeting Upload dialog and allows you to upload meeting related files in the event that the automatic upload fails.

 See memetic
Manage Stencils... (keyboard shortcut:
Alt-T-S)		 Manage Stencil Sets. This opens a dialog which allows you to create / edit  and delete stencil sets and their items. See Stencils
Open Stencil (keyboard shortcut:
Alt-T-E)		 Access a submenu holding a list of all currently available stencil sets. See Stencils
Manage Link Groups (keyboard shortcut:
Alt-T-G)		 Manage Link Groups. This opens a dialog which allows you to create / edit  and delete link groups and their link types. See Link Groups
Restore Default Link Group (keyboard shortcut:
Alt-T-R)		 Restore the default IBIS link group. You can edit the default IBIS link group through the Link Group Management dialog. If at any time you wish to restore the original setting for this group, you can use this button to do so. See Link Groups
Scribble Pad      (keyboard shortcut:
Alt-T-A)

This opes a submenu with the following Scribble Pad options:

  • Activate Scribble Pad
  • De-activate Scribble Pad
  • Save Scribble Pad
  • Clear Scribble Pad
See Scribble Pad

Reset Toolbars to Default
(keyboard shortcut:
Alt-T-R)

If you have moved your toolbars around to various side of the main screen, or reordered them, selecting this will reset them all to their default positions

See Toolbars
Project Options...
(keyboard shortcut:
Alt-T-P)

This opens the Project Options dialog (Project Preferences on the Mac).

Through this dialog you can set the node formatting defaults for newly created nodes.

See Project Options

User Options...
(keyboard shortcut:
Alt-T-U)

This opens the User Options dialog (User Preferences on the Mac).

Through this dialog you can set a vriety of options store on a per Compendium installation basis

See User Options

Focus the Desktop Area
(keyboard shortcut:
Alt-T-F) or F11

This option places the keyboard input focus on the main desktop area.

  

Focus the Left Tab Area
(keyboard shortcut:
Alt-T-C) or F12

This option places the keyboard input focus on the left tab area.
