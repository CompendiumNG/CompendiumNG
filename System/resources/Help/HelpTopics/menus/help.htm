Menu option

For more information

Welcome
(keyboard shortcut:
Alt-H-W)

Open the Welcome page

 

Help Contents
(keyboard shortcut:
Alt-H-H)

launch the help window.

 
Online Help Movies (keyboard shortcut: Alt+H+M) Launch a web browsers to view the online help movies web page for Compendium  
Quick Reference (pdf) (keyboard shortcut: Alt+H+R) Open a pdf file with a quick reference guide to Compendium.  
Quick Start Movie (html) (keyboard shortcut: Alt+H+Q) Open a web browser and view the Compendium 'Quick Start' help movie html page.  

Online Bug Reporting
(keyboard shortcut: Alt+H+B)

Takes you to the Compendium Reporting System website.

 

Online Release Notes
(keyboard shortcut:
Alt-H-N)

Takes you to the Release Notes Page for this version of Compendium.

 

Online Known Issues
(keyboard shortcut:
Alt-H-K)

Takes you to the the Compendium Reporting System website with a search for the keyword 'Known Issue'.

 

About
(keyboard shortcut:
Alt-H-A)

launch the About dialog which has version information and various useful links.

 

Help On Item
(keyboard shortcut:
Alt-H-I)

Allows context sensitive help access for some items.
