The Node Format Toolbar

This toolbar is only available for Map views at present.

Figure 1. The Node Format Toolbar


Toolbar option

Allows you to
  Detail Indicator

show / hide the node detail indicator. If you choose to display this indicator it will only appear if you have any detail text in the node contents.

It will be drawn in the top right-hand corner of the node icon area and be displayed as a ' * ' . If you roll your mouse over this it will preview the details text. If you click your mouse on it, it will open the node Contents dialog.

  Weight Indicator

show / hide the node weight indicator. If you choose to display this indicator it will only appear if the node is a map or list with nodes inside.

This will be displayed in the bottom left-hand corner of the node icon area and it will be a number which will indicator the number or nodes the view contains.

 Tags Indicator

show / hide the node tags indicator. If you choose to display this indicator it will only appear if the node has had one or more tags assigned.

This will be displayed in the top left-hand corner of the node icon area and it will be a ' T '. If you roll your mouse over it, it will display a list of the tags which have been added. If you click it, it will open the Tags dialog.

 Transclusion Indicator

show / hide the node transclusion indicator. If you choose to display this indicator it will only appear if the node is in more than one view.

This will be displayed in the bottom right-hand corner of the node icon area and it will be a number which will indicator the number or views the node appears in. If you roll your mouse over it, it will open a list of the view that the node is in. This list is interactive. You can click a view on the list and that view will be opened and the appropriate node highlighted.

  Large / Small Icon toggle the size of the icons on the selected nodes between the standard size (32 x 32 pixels) and the smaller size (16 x 16 pixels).
  hide / Show icon toggle the icons on the selected nodes between being visible and not visible.
