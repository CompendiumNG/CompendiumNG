Using the Trashbin


Below is an example of the trashbin dialog:


Figure 1. The Trashbin Dialog


There are five main buttons down the right-hand side of the dialog window:


Button

Purpose

Purge

Purges and selected nodes from the database, perminantely removing them.

Close

Closes the Trashbin dialog.

Select All

Select all the nodes in the Trashbin list.

Restore Here

Restore any selected nodes to the User's Home Map.

Restore to View

This will open another dialog (see figure 2. below) containing a list of active views that the selected node has been deleted from. You can then retore the node to one or more of those views


Figure 2. The 'Restore to Views' Dialog


N.B. This will only work for one node at a time. If you select more than node node before pressing the button, it will take the first one.
