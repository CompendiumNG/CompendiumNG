Project Options (Preferences)


The project options (project preferences on the Mac), dialog can be located on the Tools Menu.

 It currently has three tabs:


Node Text Tab


Figure 1. The Node Text tab.

There are two sections to this window. In the top section you can select the font face, font size and font style, which you wish to use for newly created nodes nodes and for all list nodes. The results can be seem in the preview pane.

In the bottom section you can set the default node label wrap width for new nodes on maps, which determines after how many characters the text of a node label wraps. It always wraps on whole words where possible, so this figure may not always be applied exactly.

Below this is a second option called 'Auto-open Node Detaild for Long Labels''. If you select this option, then, when editing a node label, after a set number of character (specified in the field below - default is 100), the node contents dialog will automatically popup and the text entered will be added into the details box.


Node Extras Tab

The second tab is called 'Node Extras', (see figure 2):


Figure 2. The Node Extras tab.


Option

Purpose

Show tag indicator

If you check this, then all newly created nodes with display the tags indicator ('T') in the top left corner of the node icon.

This will only display if one or more tags has been added to the node. Rolling over the indicator will display a list of the tags added. Clicking the indicator will open the Tags dialog.

Show text indicator

If you check this, then all newly created nodes with display the text indicator
(' * ') in the top right corner of the node icon.

This will only display if the node has additional detail test. Rolling over this indicator will preview the text. Clicking the indicator will open the node Contents dialog.

Show transclusion indicator

If you check this, then all newly created nodes with display the transclusion indicator number in the bottom right corner of the node icon.

This number will indicate the number of parent views that the node appears in. Rolling over this indicator will display a list of the parent views for the node. Clicking an item on the list will open that view and highlight the node.

Show weight indicator

If you check this, then all newly created nodes with display the weight indicator number in the bottom left corner of the node icon.

This number only appears on map and list nodes and idicates the number of nodes that the view conatins.

Use small icon If you check this, then all newly created nodes will use the smaller node icon will be displayed (16 x 16 pixels) rather than the stadard size one (32 x 32 pixels).
Hide icon If you check this, then all newly created nodes will not display an icon at all.
Show Map Border when map has image

If you check this, then newly created view nodes will display a border around them when the standard icon is replaced by an external image. This is to help distinguish view nodes with images from reference nodes with images.


Linked Files Tab

The third tab holds Linked Files folder location settings, (see figure 3 below):

Figure 2. The Linked Files tab.


The default location for your linked files is in the 'Linked Files' folder inside the Compendium folder. However you can specify a new location here. Full instructions can be seen in the screen texts.
