Cascade
(keyboard shortcut:
Alt-W-C)

Arrange all the open view windows in a cascade.

This resizes all open windows quite small, and overlaps them slightly, moving diagonally from left to right down the screen.

 

Tile
(keyboard shortcut:
Alt-W-T)

This tiles all the currently open windows.

  

Expand All
(keyboard shortcut:
Alt-W-E)

Expand all currently open view windows.

Note: This maximises all open views. If you then un-maximise the top window, all windows below are automatically un-maximised.

  

Close All
(keyboard shortcut:
Alt-W-A)

Close all currently open view windows.

Note: After using this command, you can use the Find a Map/List option on the Map menu to open a view.

  

... Open Windows list

Below the 'Close All' menu option there will be a list of any open views. Selecting one of these, will bring that view to the front. The number in front of the view title represents the window order, with 0 being the currently active window
